# Travely MVC

Web đặt tour du lịch PHP MVC chạy trên XAMPP, dùng PDO MySQL.

## Chức năng chính

- Người dùng: xem tour, tìm kiếm/lọc tour, đăng ký, đăng nhập, yêu thích tour, đặt tour và xem booking cá nhân.
- Admin: dashboard thống kê, quản lý tour, booking, liên hệ, người dùng và phân quyền.
